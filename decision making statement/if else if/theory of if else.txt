The if-else-if ladder statement executes one condition from multiple statements.
that is first true condition.

syntax:

if(boolean expression)
{
body statement;
}
else if(boolean expression)
{
body statement;
}
else if(boolean expression)
{
body statement;
}
else
{
body statement;
}
